
## 词类
两个句子组成方式相同，比如都是主语+谓语动词+宾语，
可是句中包含的词的类型并不一定相同。
```
The rabbit ate a carrot.
He saw something over there.

句型相同但是包含的词不同，也就是说同一类句子成分可能有不同的词类。
```

### 2.冠词 Articles
### 3.代词 Pronouns
### 4.形容词 Adjectives

### 7.介词 Prepositions
```

```
### 8.叹词 Interjections
```
Ah,the carrot is tasty.
```
